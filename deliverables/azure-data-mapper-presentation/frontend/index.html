What to remember first

It is Standard-only authoring

Microsoft documents Data Mapper authoring in Visual Studio Code inside Azure Logic Apps Standard projects.

Two files matter

The visual map is saved as .lml and the executable transform is generated as .xslt.

The action is specific

Maps created with Data Mapper are called with Transform using Data Mapper XSLT, not the older Transform XML action.

Recommended path

Use this site as a short walkthrough rather than a reference dump. The pages follow the sequence a team usually needs.
