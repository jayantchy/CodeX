Which file is the visual Data Mapper design source?

The .lml file is the visual authoring source. The generated .xslt is the runtime transform.

Which workflow action calls a Data Mapper-created map?

Data Mapper maps are called through the Data Mapper Operations action.

What should a developer do before testing a map locally?

Saving matters because the mapper test should run against the current generated XSLT.

Which item should not be committed to Git?

local.settings.json can contain local secrets and machine-specific values.

What is a documented Data Mapper risk area?

Microsoft Learn calls out Filter string comparison behavior as a current limitation.

Which function is the best workaround for unreliable Filter string comparisons?

Execute XPath lets you write item[name='value'] predicates — more reliable than the visual Filter function for string matching.

Where should custom XSLT files for Run XSLT be placed?

The Data Mapper scans Artifacts/DataMapper/Extensions/InlineXSLT/ to populate the Run XSLT function dropdown.
